Instructions

Colors
The colors button allow you to add a new color to the palette, please note that you can add max 12 colors, after that every time that you add a new one, one of the current colors in your palette will be replaced
Bucket tool
Use the Bucket tool when you want to fill the grid with the same color
Grid size
Use the grid select option if you want to change the size of your grid
